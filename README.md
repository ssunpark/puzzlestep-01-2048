# puzzlestep-01-2048

알고리즘 기반 퍼즐 게임 구현 시리즈의 첫 번째 프로젝트입니다. 
2048 게임의 핵심 로직을 직접 설계하며 그리드 데이터 관리와 효율적인 타일 이동 알고리즘을 익히는 것을 목적으로 합니다.

## 학습 목표
- 2차원 배열(Grid)을 활용한 데이터 관리
- 타일 병합(Merge) 및 이동(Move) 로직의 예외 처리
- 입력 방향에 따른 데이터 업데이트 최적화

## 핵심 알고리즘 및 구조
- **Grid System**: 4x4 배열을 기반으로 타일의 위치와 상태 관리
- **Move Logic**: 각 행/열 단위의 스택 또는 리스트 조작을 통한 병합 처리
- **State Check**: 이동 가능 여부 판별 및 게임 오버/클리어 상태 체크

## 구현 사항
- [ ] 4x4 그리드 생성 및 랜덤 타일 스폰
- [ ] 상하좌우 방향별 타일 이동 및 병합 로직
- [ ] 점수 계산 시스템
- [ ] 게임 오버 및 재시작 기능
